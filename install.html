Installation Protocol

Official Distribution Guide v1.1

01

Secure Download

Click the "Download APK" button on the homepage. If your browser displays a "File might be harmful" warning, select "Download anyway". This is a standard notification for files not hosted on the Play Store.

02

System Authorization

Allow Installation from Browser

Go to Settings -> Allow from this source to enable the system-level installer. This is required for the application to manage its own security updates.

03

Play Protect Verification

When the prompt appears, tap "More details" and then select "Install anyway". Every NoRelapse build is VirusTotal scanned and verified safe.

info Technical Transparency

NoRelapse requires AccessibilityService and Request Install Packages permissions to maintain real-time "Guardian Shield" protection and secure, direct self-updates.
