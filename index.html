Privacy Architecture

ZERO-TRANSMISSION PROCESSING

NoRelapse analyzes your browsing activity entirely within your device's local memory. The pattern-matching engine, the keyword filters, the detection logic — all of it executes on-chip. Nothing is written to a remote database. Nothing is transmitted over a network. When the session ends, the analysis is purged. Your private struggle belongs to no one but you.

ARCHITECTURE: Local-only inference engine. No API calls. No telemetry. No usage logs.
Enforcement Layer

ACCESSIBILITY SERVICE ENFORCEMENT

Standard app blockers operate inside the browser. They are peers to the threat — and the threat can route around them. NoRelapse operates at the AccessibilityService API, which sits one layer beneath every browser and app on your device. When a blocked URL loads in any browser — Chrome, Firefox, Brave, any private window — NoRelapse intercepts and terminates. There is no workaround because there is no gap.

LAYER: AccessibilityService API. Covers all installed browsers. Real-time interruption.
